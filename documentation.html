Documentation

Introduction

Welcome to the documentation of your Login and Registration template. This template can be easily intergrated into your project.

Features

This template was designed with functionality in mind. Here is a list of features found in this template:

  • It is responsive as it is built on the Bootstrap version 5 framework.
  • It has a datepicker to help with the selection of a users date of birth upon registration.
  • During registration, a user can toggle whether they want to show or hide their password with a checkbox.
  • Font Awesome has been used to make the buttons more interactive.
  • There is a fullscreen background image that can be changed by altering the custom stylesheet and images folder in the assets folder

How to add the template to your project

Step 1

Copy the assets folder into your projects file structure. This is important because this folder contains the relevant stylesheets, JavaScript, Jquery and the background image.

Step 2

Copy and paste the HTML pages into your project. Please note that you will have to change the name of the login template from 'index.html' to 'login.html' or to a name of your liking according to your project.

Step 3

Connect the template to a server side language, this could be PHP, NodeJS or any language that supports HTML. Here is a list of YouTube videos that could aid with connecting this template to a server side language:
  • How to connect an HTML form with MySQL Database using PHP
  • A simple signup and login system with PHP and Mysql database
  • Login and Signup tutorial with nodejs, express & mongodb
